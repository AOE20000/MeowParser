MeowParser Version Information
==============================

Current Version: 2.2.0
Release Date: 2024-XX-XX
Status: Stable

Version History:
----------------

v2.2.0 (Current)
- Config File System: Multiple configuration files support
- Config Switching: Quick switch between different scenarios
- Rule Groups: UI organization only, no execution order impact
- Rule Order: Rules execute in file order, deterministic
- Config Import/Export: Easy sharing and backup
- Auto Migration: Automatic migration from old format
- Bug Fixes: Fixed 8 bugs in rule editor

v2.1.0
- Rule Groups: Create and manage multiple rule groups
- Independent enable/disable for each group
- Improved rule management UI
- Better organization for different scenarios

v2.0.0
- Complete rewrite using PyQt6
- Cross-platform support (Windows/Linux/macOS)
- Improved focus management
- Modern UI with dark theme
- Window manager with search and filter
- Replacement rule editor with regex support
- Debug window with logging
- Single instance protection

v1.0.0 (tkinter) - Deprecated
- Initial release (Windows only)
- Replaced by PyQt6 version

Platform Support:
-----------------
- Windows: Full support ✅
- Linux: 90% complete 🚧
- macOS: Untested ⏳
