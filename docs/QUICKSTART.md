# MeowParser 快速开始指南

**版本：** 2.2.0

---

## 1. 安装

### 下载项目

```bash
git clone https://github.com/yourusername/MeowParser.git
cd MeowParser
```

### 安装依赖

```bash
pip install -r requirements_pyqt6.txt
```

**依赖列表：**
- PyQt6 >= 6.6.0
- keyboard
- psutil
- pywin32 (Windows)

---

## 2. 运行

### Windows

```bash
# 方式1：双击运行
run.bat

# 方式2：命令行
python meow_parser.py
```

### Linux/macOS

```bash
# 方式1：脚本运行
chmod +x run.sh
./run.sh

# 方式2：命令行
python3 meow_parser.py
```

⚠️ **注意：需要管理员/root 权限**

---

## 3. 首次使用

### 启动程序

1. 运行程序后，系统托盘会出现"喵"字图标
2. 图标颜色：
   - 🔴 红色 = 禁用状态
   - 🟢 绿色 = 启用状态

### 添加窗口到白名单

1. 右键托盘图标
2. 选择"窗口管理"
3. 找到你要使用的应用窗口
4. 双击窗口名称启用

### 启用功能

1. 右键托盘图标
2. 选择"启用/禁用"
3. 图标变为绿色表示已启用

---

## 4. 基本使用

### 触发悬浮窗

1. 在白名单窗口中点击输入框
2. 输入任意内容激活
3. 按**空格键**触发悬浮窗

### 输入文本

1. 在悬浮窗中输入内容
2. 按**回车**：应用规则并发送
3. 按**Ctrl+回车**：发送原始内容
4. 按**ESC**：取消输入

---

## 5. 配置规则

### 打开配置编辑器

1. 右键托盘图标
2. 选择"替换规则"

### 创建新配置

1. 点击"新建"按钮
2. 输入配置名称（如"工作场景"）
3. 输入描述（可选）

### 添加规则组

1. 点击"添加规则组"
2. 输入组名（如"常用替换"）

### 添加规则

1. 双击规则组
2. 在规则列表中添加规则：
   - **描述**：规则说明
   - **匹配文本**：要替换的文本
   - **替换为**：替换后的文本
   - **正则表达式**：是否使用正则

### 示例规则

**文本替换：**
- 匹配：`我`
- 替换：`喵`

**正则替换：**
- 匹配：`\d+`
- 替换：`很多`
- 勾选"使用正则表达式"

## 9. 常见问题

### 程序无法启动

**原因：** 缺少管理员权限

**解决：**
- Windows：右键 → 以管理员身份运行
- Linux：使用 `sudo`

### 悬浮窗不出现

**原因：** 窗口未添加到白名单

**解决：**
1. 打开窗口管理
2. 找到目标窗口
3. 双击启用

### 替换不生效

**原因：** 规则未启用或配置错误

**解决：**
1. 检查规则是否勾选"启用"
2. 检查匹配文本是否正确
3. 查看调试日志
