# MeowParser 主题切换 - 快速指南

## 🚀 快速开始（30秒）

### 1. 启动程序

```bash
python meow_parser.py
```

### 2. 切换主题

```
右键托盘图标 "喵" 
  ↓
选择 "主题"
  ↓
选择你想要的模式：
  🌓 自动（跟随系统）← 推荐
  🌙 深色
  ☀️ 浅色
```

### 3. 完成！

主题已切换，享受新的视觉体验！

---

## 💡 三种模式对比

| 模式 | 说明 | 适用场景 |
|-----|------|---------|
| 🌓 自动 | 跟随系统主题自动切换 | 日常使用（推荐） |
| 🌙 深色 | 固定使用深色主题 | 夜间工作 |
| ☀️ 浅色 | 固定使用浅色主题 | 白天工作 |

## ⚙️ 配置文件

主题设置保存在：`.meowparser/theme_settings.json`

```json
{
  "theme": "auto"
}
```

可选值：`"auto"` | `"dark"` | `"light"`

---

## 🔧 系统主题设置

### Windows 10/11

```
设置 → 个性化 → 颜色 → 选择模式
```

### macOS

```
系统偏好设置 → 通用 → 外观
```

### Linux (GNOME)

```
设置 → 外观 → 样式
```

## ❓ 常见问题

**Q: 如何知道当前使用的是哪个主题？**  
A: 查看托盘菜单中的勾选项，或观察界面颜色。

**Q: 自动模式不工作？**  
A: 确保系统支持主题检测，或手动选择主题。

**Q: 切换后窗口没更新？**  
A: 关闭窗口后重新打开即可。
