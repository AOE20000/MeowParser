# 主题切换使用示例

## 快速切换主题

### 方法 1：托盘菜单（推荐）

```
1. 右键点击托盘图标 "喵"
   ↓
2. 选择 "主题" 菜单
   ↓
3. 选择主题模式：
   - 🌓 自动（跟随系统）
   - 🌙 深色
   - ☀️ 浅色
```

### 方法 2：编辑配置文件

编辑 `.meowparser/theme_settings.json`：

```json
{
  "theme": "auto"
}
```

然后重启 MeowParser。

## 自动模式工作原理

```
启动 MeowParser
    ↓
检测系统主题
    ↓
应用对应主题
    ↓
启动监控定时器（每 5 秒）
    ↓
检测系统主题变化
    ↓
如有变化，自动切换主题
```

## 常见问题

### Q: 如何知道当前使用的是哪个主题？

A: 查看以下位置：
1. 托盘菜单 → 主题 → 查看勾选项
2. 观察界面颜色（深色/浅色）
3. 查看 `.meowparser/theme_settings.json`

### Q: 自动模式下如何强制刷新？

A: 
1. 切换到其他主题模式
2. 再切换回自动模式
3. 或重启 MeowParser

### Q: 可以设置不同窗口使用不同主题吗？

A: 目前不支持，所有窗口使用统一主题。

## 最佳实践

### 1. 使用自动模式

**推荐理由：**
- 自动适应环境
- 无需手动切换
- 保护视力健康

### 2. 配合系统主题

**Windows 10/11：**
```
设置 → 个性化 → 颜色 → 选择模式
```

**macOS：**
```
系统偏好设置 → 通用 → 外观
```

**Linux (GNOME)：**
```
设置 → 外观 → 样式
```

### 3. 定时切换系统主题

**Windows：**
- 使用"夜间模式"自动切换
- 设置 → 系统 → 显示 → 夜间模式

**macOS：**
- 使用"自动"模式
- 系统偏好设置 → 通用 → 外观 → 自动

**Linux：**
- 使用 GNOME Tweaks 或类似工具
- 设置定时任务切换主题
