# MeowParser v2.3.0 - 主题切换功能更新

**发布日期：** 2026-03-30  
**版本：** v2.3.0

---

## 🎉 主要更新

### 自动主题切换

MeowParser 现在支持自动跟随系统主题切换深色/浅色模式！

**三种主题模式：**
- 🌓 **自动模式**：跟随系统主题自动切换（推荐）
- 🌙 **深色主题**：固定使用深色主题
- ☀️ **浅色主题**：固定使用浅色主题

---

## 🚀 快速开始

### 切换主题

1. 右键点击系统托盘中的"喵"图标
2. 选择"主题"菜单
3. 选择你想要的主题模式

### 自动模式

选择"自动（跟随系统）"后，MeowParser 会：
- 自动检测你的系统主题设置
- 系统是深色模式时，使用深色主题
- 系统是浅色模式时，使用浅色主题
- 每 5 秒自动检查系统主题变化

---

## 💡 使用场景

### 推荐配置

**白天工作：**
- 系统设置为浅色模式
- MeowParser 自动切换到浅色主题
- 清新明亮，适合明亮环境

**夜间使用：**
- 系统设置为深色模式
- MeowParser 自动切换到深色主题
- 减少眼睛疲劳，保护视力

**固定偏好：**
- 如果你总是喜欢深色或浅色
- 可以手动选择固定主题
- 不受系统主题影响

## 🔧 技术实现

### 系统主题检测

#### Windows 10/11

读取注册表：
```
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Themes\Personalize
键名：AppsUseLightTheme
值：1 = 浅色，0 = 深色
```

#### macOS

读取系统设置：
```bash
defaults read -g AppleInterfaceStyle
# 返回 "Dark" = 深色
# 返回错误 = 浅色
```

#### Linux

读取 GTK 配置：
```
~/.config/gtk-3.0/settings.ini
[Settings]
gtk-theme-name=Adwaita-dark  # 包含 "dark" = 深色
```

### 自动监控

在自动模式下，程序每 5 秒检查一次系统主题：
- 检测到主题变化时自动切换
- 无需重启应用
- 实时响应系统设置

## 📁 配置文件

主题偏好保存在：`.meowparser/theme_settings.json`

```json
{
  "theme": "auto"
}
```

**可选值：**
- `"auto"` - 自动模式（推荐）
- `"dark"` - 深色主题
- `"light"` - 浅色主题

## 🐛 已知问题

### 主题切换后部分窗口未更新

**问题：** 切换主题后，已打开的窗口（如配置编辑器）可能不会立即更新。

**解决方案：** 关闭窗口后重新打开即可。

### Linux 主题检测不准确

**问题：** 某些 Linux 桌面环境可能无法正确检测主题。

**解决方案：** 手动选择深色或浅色主题。
