## [2.2.0] - 2024-XX-XX

### 新增
- ✨ 配置文件系统：支持多个配置文件，每个配置文件是一个完整的规则集
- ✨ 配置文件切换：通过下拉菜单快速切换不同场景的配置
- ✨ 规则组折叠：规则组仅用于UI组织，不影响规则执行顺序
- ✨ 规则顺序确定：规则按配置文件中的顺序依次执行
- ✨ 配置导入导出：方便分享和备份完整配置
- ✨ 自动迁移：自动迁移旧版本配置到新格式

### 改进
- 🚀 更简单的概念模型：配置文件 = 规则集
- 🚀 更清晰的配置结构：规则组仅用于UI折叠
- 🚀 更确定的执行顺序：规则按文件顺序执行，无歧义
- 🚀 更方便的场景切换：不同场景使用不同配置文件
- 🚀 更容易的配置分享：直接分享配置文件
- 📝 配置文件位置：`.meowparser/rules/` 目录

### 变更
- 🔄 配置文件格式重新设计：支持配置元数据和规则组列表
- 🔄 规则组不再影响执行顺序：仅用于UI组织
- 🔄 配置管理方式改变：从单一配置到多配置文件
- 🔄 替换规则编辑器完全重构：支持配置文件管理

### 修复
- 🐛 修复规则组顺序不确定的问题
- 🐛 修复配置迁移时引用断开的问题
- 🐛 修复删除规则组后状态混乱的问题
- 🐛 修复重命名后未保持选中状态的问题
- 🐛 修复允许添加空替换规则的问题
- 🐛 修复删除规则缺少确认对话框的问题
- 🐛 修复缺少正则表达式验证的问题
- 🐛 修复导入时变量重用的问题

## [2.1.0] - 2024-XX-XX

### 新增
- ✨ 规则组功能：支持创建多个规则组，分类管理不同场景的替换规则
- ✨ 规则组独立启用/禁用：每个规则组可以独立控制
- ✨ 双击切换：双击规则组可快速切换启用状态
- ✨ 规则组管理：支持新建、重命名、删除规则组
- ✨ 导入/导出规则组：方便分享和备份规则组
- ✨ 内置规则组：默认包含"喵语转换"规则组

### 改进
- 🚀 更灵活的规则管理：可以为不同场景创建不同的规则组
- 🚀 更好的用户体验：左右分栏布局，规则组和规则分开管理
- 🚀 完全可配置：所有文本处理逻辑都通过规则组管理，无硬编码
- 🚀 规则组分享：可以导出规则组分享给其他用户
- 📝 配置文件格式升级：支持规则组结构（自动迁移旧格式）

### 变更
- 🔄 配置文件格式从单一规则列表改为规则组结构
- 🔄 替换规则编辑器UI重新设计
- 🔄 移除硬编码的"喵"字添加逻辑，改为"喵语转换"规则组
- 🔄 默认配置简化为"喵语转换"规则组
