# 更新日志

## [2.3.0] - 2026-03-30

### 新增
- 🌓 **自动主题切换功能**
  - 自动模式：跟随系统主题自动切换深色/浅色
  - 深色主题：经典的 Breeze Dark 风格
  - 浅色主题：清新的 Breeze Light 风格
  - 支持 Windows、macOS、Linux 系统主题检测
  - 每 5 秒自动检查系统主题变化
  - 保存用户主题偏好到 `.meowparser/theme_settings.json`
  - 详见 [主题切换指南](docs/THEME_GUIDE.md)

- 🎨 **浅色主题设计**
  - 完整的 Breeze Light 样式
  - 与深色主题一致的交互体验
  - 优化的颜色对比度
  - 适合明亮环境使用

- 🔄 **主题监控系统**
  - 实时检测系统主题变化
  - 自动应用对应主题
  - 可手动切换主题模式

### 改进
- 🎨 重构样式管理系统
  - 创建 `ThemeDetector` 类处理系统主题检测
  - 创建 `StyleManager` 类统一管理主题
  - 支持主题偏好持久化
- 🖱️ 托盘菜单新增主题切换选项
  - 🌓 自动（跟随系统）
  - 🌙 深色
  - ☀️ 浅色
- 🔒 **自动权限提升**
  - Windows: 自动弹出 UAC 对话框请求管理员权限
  - Linux/macOS: 自动使用 sudo 请求 root 权限
  - 权限被拒绝时显示友好的错误提示
- 🎨 简化托盘菜单
  - 移除权限状态显示（已在启动时强制检查）
  - 移除快捷键提示（简化界面）
- 📝 更新文档
  - 新增主题切换指南
  - 新增权限提升说明
  - 更新 README 功能说明

### 修复
- 🐛 修复 `show_message` 方法参数类型错误
- 🐛 修复托盘右键菜单不跟随主题变化的问题
- 🎨 增强浅色主题列表字体对比度（达到 WCAG AAA 标准）
- ✨ 菜单样式现在会根据当前主题动态更新
- 🔧 改进浅色主题的树形控件可读性

### 技术细节
- Windows: 读取注册表 `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Themes\Personalize\AppsUseLightTheme`
- macOS: 读取 `defaults read -g AppleInterfaceStyle`
- Linux: 读取 GTK 配置 `~/.config/gtk-3.0/settings.ini`

- 🚀 更清晰的配置结构：规则组仅用于UI折叠

- 🚀 更方便的场景切换：不同场景使用不同配置文件
- 🚀 更容易的配置分享：直接分享配置文件
