# AutoMeowEV 自动发喵程序

一个创新的Windows系统托盘应用，通过智能输入监听和悬浮窗实现文本替换和自动发送"喵"。

## 功能特点

- 系统托盘运行，占用资源少
- 红/绿图标显示启用状态
- 智能输入监听，自动弹出悬浮窗
- 悬浮输入窗口，不干扰原有输入框
- 智能文本替换功能（支持全文匹配替换）
- 单实例运行保护
- 窗口白名单管理

## 工作原理

启用后，当你在白名单窗口中开始输入时：
1. 程序监听你的键盘输入
2. 当你激活输入框（点击输入框或输入任意内容）后，按空格键时自动弹出悬浮窗
3. 原输入框的内容保持不变，不会被删除
4. 在悬浮窗中输入你要发送的文本
5. 按回车：应用替换规则或添加"喵"，然后自动输入到原位置
6. 按Ctrl+回车：直接发送原始内容，不做任何替换
7. 按ESC：取消输入，关闭悬浮窗

注意：悬浮窗仅在"激活输入框"后按空格键时触发，确保不会误触发。

## 使用方法

1. 运行程序后会在系统托盘显示一个"喵"字图标
2. 图标颜色表示状态：
   - 红色：禁用状态
   - 绿色：启用状态
3. 右键点击图标可以：
   - 启用/禁用功能
   - 打开窗口管理（选择启用的窗口）
   - 打开替换规则（配置文本替换）
   - 查看权限状态
   - 退出程序
4. 在窗口管理中：
   - 可以搜索窗口
   - 双击窗口切换启用状态
   - 可随时刷新窗口列表
5. 在替换规则中：
   - 可以添加、编辑、删除替换规则
   - 支持调整规则优先级（上移/下移）
   - 可以启用/禁用整个替换功能
   - 规则按顺序匹配，匹配到的文本会被替换
6. 权限状态显示：
   - ✓ 管理员：可以正常使用所有功能
   - ✗ 普通用户：功能受限，需要以管理员身份运行

## 快捷键

- **回车键**：触发文本替换或添加"喵"，并自动发送
- **Ctrl + 回车**：发送原始内容，不进行任何替换
- **ESC键**：取消输入，关闭悬浮窗
- **Ctrl + Shift + M**：打开调试窗口（用于测试和查看日志）

## 调试功能

程序内置了调试窗口，方便测试和排查问题：

1. 按 `Ctrl+Shift+M` 或右键托盘图标选择"调试窗口"打开
2. 调试窗口包含两个部分：
   - **日志输出**：显示程序运行的详细日志
   - **测试输入框**：此输入框默认在白名单中，可直接测试替换功能
3. 日志功能：
   - 查看文本处理过程
   - 查看焦点恢复状态
   - 查看错误信息
   - 可清空或复制日志

## 文本替换功能

当按下回车键时：
- 程序会扫描悬浮窗中的文本，将所有匹配的词替换为指定内容
- 支持多个规则同时生效，按优先级顺序依次替换
- 如果没有匹配到任何规则，会在句尾添加"喵"
- 每个规则会替换文本中所有匹配的位置

示例：
- 原句："这是我的一句话，他即将被AutoMeow替换"
- 规则："我" → "喵"，"他" → "喵"
- 结果："这是喵的一句话，喵即将被AutoMeowEV替换"

常用规则示例：
- "我" → "喵"
- "你" → "喵"
- "他" → "喵"
- "她" → "喵"

## 注意事项

- 需要管理员权限运行（因为需要键盘监听）
- 启用状态下会监听键盘输入
- 建议在聊天软件等适当场合使用
- 悬浮窗会自动置顶，输入完成后自动隐藏
- 程序会自动记录鼠标位置，用于点击目标输入框
